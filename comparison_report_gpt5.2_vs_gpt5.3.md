## 📈 关键发现

### 1. GPT-5.3 全面碾压 GPT-5.2

经重新测试后，GPT-5.3 在**所有四档**的**所有核心维度**（延迟、TTFT、输出量、TPS）均优于 GPT-5.2：
- **延迟**：GPT-5.3 全面更低，改善幅度 15.1% ~ 32.0%。
- **TTFT**：GPT-5.3 全面更快，改善幅度 17.8% ~ 48.9%。
- **TPS**：GPT-5.3 全面更高，提升幅度 21.2% ~ 107.8%。
- **输出 Tokens**：GPT-5.3 全面更多，增幅 34.0% ~ 61.7%。

### 2. GPT-5.2 重测后延迟普遍上升

与上次测试相比，GPT-5.2 本次延迟有所增加：
- Low: 17,629 → 19,780 ms (+12.2%)
- Medium: 16,992 → 24,316 ms (+43.1%)
- High: 50,218 → 48,691 ms (-3.0%，基本持平)
- xhigh: 47,530 → 75,654 ms (+59.2%)

这可能受服务端负载波动影响，也说明性能测试需多时段采样。

### 3. High 档 TTFR 唯一亮点

GPT-5.2 仅在 High 档 TTFR 上以 10,052 ms vs 14,422 ms 胜出（快 30.3%），在其余所有档位和维度上均不敌 GPT-5.3。

### 4. xhigh 档 TPS 仍然异常

- GPT-5.2 xhigh TPS 499 tokens/s，GPT-5.3 xhigh TPS 1,036 tokens/s。
- 高 TPS 源自长时间推理后批量输出，不代表持续吞吐能力。
