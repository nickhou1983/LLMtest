## 📈 关键发现

### 1. GPT-5.3 在输出量与吞吐上全面领先

- **所有四档**输出 Tokens 和 TPS 均显著高于 GPT-5.2。
- xhigh 档差距最大：输出量 +128.6%，TPS +641.8%。
- 这意味着 GPT-5.3 在相同推理强度下能生成更完整、更丰富的代码。

### 2. 延迟表现各有胜负

- **Low / High**：GPT-5.3 延迟更低（分别 -4.8%、-34.1%）。
- **Medium / xhigh**：GPT-5.2 延迟更低（分别 -17.5%、-30.5%）。
- GPT-5.3 在 xhigh 档延迟明显更高（62s vs 47.5s），输出也更多（4,067 vs 1,779 tokens），说明额外时间主要花在生成更多内容上。

### 3. TTFT 表现

- **Low / Medium / High 三档**：GPT-5.3 首 Token 更快（改善 14.8% ~ 44.2%）。
- **xhigh 档**：GPT-5.2 首 Token 更快（33.6s vs 55.2s），GPT-5.3 思考时间更长。

### 4. xhigh 档 TPS 异常高值

- GPT-5.3 xhigh 平均 TPS 达 1,036 tokens/s，单次最高 1,947 tokens/s。
- 这可能因为长时间思考后批量输出 token，导致瞬时 TPS 极高。
